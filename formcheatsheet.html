HTML Form reference

Form Tags

The <form> element is used to create an HTML form for user input.

The <form> element can contain one or more of the following form elements:

Tag Name Description
<input> Input Specifies an input field where the user can enter data.
<textarea> Textarea Defines a multi-line text input control.
<button> Button Defines a clickable button
<select> Select Used to create a drop-down list
<option> Option Defines an option in a select list
<optgroup> Optgroup Used to group related options in a drop-down list
<fieldset> Fieldset Used to group related elements in a form
<label> Label Defines a label for several elements
<output> Output Represents the result of a calculation

Form Attributes

Attribute Name Description
<action> Action Specifies where to send the form-data when a form is submitted
<method> Method Specifies the HTTP method to use when sending form-data
<name> Name Specifies the name of a form
<target> Target Specifies where to display the response that is received after submitting the form
